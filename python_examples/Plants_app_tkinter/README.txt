This GUI app allows users to do CRUD operations and it shows 
different information about plants and pots with either 
generated values or values obtained trough API (like air 
temperature). All information about plants and pots (name, 
id, photo, etc.) and the changes that user makes is saved 
in a database (sqlite3) via SQL commands.
