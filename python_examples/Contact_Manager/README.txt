Treba pokrenuti 'main_Contact_Manager.py'

Aplikacija se izvršava u terminalu.
Aplikacija omogućava da se vodi evidencija organizacija i informacije o njihovim zaposlenima i klijentima.
Moguće je dodavanje nove organizacije, prikaz liste svih organizacija ili detaljni prikaz pojedine organizacije.
Također se mogu update-at sve informacije pojedine organizacije, njihovih zaposlenih ili klijenata- 
Informacije se spremaju u JSON formatu --> Organization_info.json
