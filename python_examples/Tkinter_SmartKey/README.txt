Treba pokrenuti 'customTK_SmartKey_main.py' za moderniju verziju Tkinter aplikacije 
ili 'smart_key.py' za standardnu veriju Tkintera

Aplikacija se sastoji od Tkinter/CustomTkinter i sqlite3 modula
Osnovna funkcija aplikacije je mogućnost prolaza kroz vrata zvonom ili pinom ukoliko postoji user
Admin uz popis user-a ima mogućnost dodavanja, brisanja i updateanja usera u sqlite3 databazu
U bazi su spremljeni username i unique password za svakog user-a
